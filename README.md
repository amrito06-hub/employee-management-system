# Employee Management System

A full-stack Employee Management System built using React, Node.js, Express.js, and MongoDB.

## Features

- User Login and Registration
- JWT Authentication
- Dashboard
- Employee Management
- Add and View Employees
- Employee Details
- Department Management
- Payroll and Salary Management
- Generate Payroll
- Mark Salary as Paid
- Payroll History
- Search and Filter Employees
- Search and Filter Payroll Records
- Dark Mode Support
- Responsive Design
- MongoDB Database Integration
- REST APIs

## Technologies Used

### Frontend

- React.js
- React Router DOM
- Axios
- Tailwind CSS
- Vite

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs
- dotenv
